
# PRODUCT-SERVICE

This service allows to add/remove products and categories.



## Technologies
- Java 17
- Spring Boot 3.1.2
- Spring Cloud 
- Docker
- PostgreSQL
- Open Feign
## Features
#### Current:
- Product CRUD
- ProductCategory CRUD
- communication with order-service
